Overview

Hevent is a jQuery plugin that enable pointer events and transitionend/animationend events.

The advantages of pointers events is that they ensure a support accross desktop/mobile platform and all the things in between

Pointer events

Pointer event support is a copy/paste from the jquery-pointer-events.
The supported events are:

  • pointerdown
  • pointerup
  • pointermove
  • pointercancel
  • pointerover
  • pointerout

3 additionals events are supported

  • pointerswipe
  • pointerswipeleft
  • pointerswiperight
